v1.0 Vaporware
  • Rewrite of WCECL from scratch (Feb 2026) with an ARM emulator backend; ran Solitaire like the original x86 WCECL.
  • User-space CE emulator built on mocking COREDLL.DLL. Ran several CE shells (CE 5 best), plus a heuristically reconstructed Windows Mobile 5 shell.
  • Scope collapsed: CE COM is a user-space driver, kernel API traps differ per CE version, forcing per-version strategies until it had quietly become a sloppy kernel + MMU rewrite of every CE version.
  • Died on Win32 per-process object limits and leaking APIs. Replaced by v2 — a real silicon emulator.
WCECL Vaporware
  • Windows CE Compatibility Layer (https://github.com/dz333n/wcecl), 2019 — a shim DLL repointing COREDLL.DLL APIs to desktop Win32 to run x86-built CE apps.
  • Ran Solitaire and cmd.exe (contributions from @veselcraft and @TheNNX), but only a few x86 apps — never real MIPS or ARM binaries.
